# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in npmLens, please report it by emailing **[dev@pilaton.com](mailto:dev@pilaton.com)**.

**Please do not open a public GitHub issue for security vulnerabilities.**

When reporting, please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 7 days
- **Resolution**: Depends on severity, typically within 30 days

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 3.x     | ✅ Yes    |
| 2.x     | ❌ No     |
| 1.x     | ❌ No     |

Thank you for helping keep npmLens secure!
