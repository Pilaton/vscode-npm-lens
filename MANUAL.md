# MANUAL

## DEV

Запуск разработки производится в debug режиме, запустить нужно таск "Vite dev". Правка простых .ts не вызывает hot reload, поэтому придется перезапускать расширение полностью.

## PUBLISH

1. Если нужно обновить скрин, нужно вручную убрать из заголовка `DEV` и изменить версию (посмотреть ту, что будет, можно через `release-demo`).  
   Для скрина подгоняем контейнер окна с расширением до `882x760px`, чтобы конечный скрин получился `1764x1520`. Смотреть текущие окна удобно во вкладке `Computed` в правой колонке стилей `Chrome Dev Console`.

   Скрин можно сделать прямо из `Chrome Dev Console`, найдя в html коде самый верхний `<iframe class="webview ready">`, и правой мышкой из списка выбрав `Capture node screen`.

   Сделав скрин, правим `README`

2. После внесения изменений, всё линтим.
3. Запускаем `release-demo` и проверяем как будет выглядеть `CHANGELOG`. Если ОК, генерим финишный `CHANGELOG` и вручную убираем лишний текст из шапки файла.
4. Пакуем расширение финишно
